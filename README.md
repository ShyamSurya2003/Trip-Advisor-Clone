# 🌍 TripMate — Global Travel Planner

A clean and responsive travel website built using **HTML5 and CSS3**.
TripMate helps users explore destinations, hotels, restaurants, and plan their dream trips.

---

## 🚀 Live Demo

👉 https://shyamsurya2003.github.io/Udemy_Clone/

---

## 📌 Features

* 🌐 Modern navigation bar
* 🔍 Search box for destinations
* 🌎 Popular destinations section
* 🏨 Top hotels showcase
* 🍽️ Restaurants section
* 💬 Traveler reviews
* 📱 Fully responsive design
* 🎨 Clean and modern UI
* ⚡ Built using pure HTML & CSS (no JavaScript)

---

## 🛠️ Built With

* HTML5
* CSS3
* Google Fonts (Poppins)
